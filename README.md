# Rustless
Rust version of Linux `less` command.  
Uses [tui-rs](https://github.com/fdehau/tui-rs) as the underlying tui framework. 

# Usage
Run with cargo, taking in a text file as input:
```
cargo run -- [FILENAME]
```
For help, run:
```
cargo run -- help
```
